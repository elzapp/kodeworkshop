Det neste vi trenger er en skikkelig editor. Kodeklubben legger opp til at dere skal bruke IDLE, som kommer med Python, men jeg vil anbefale Microsoft Visual Studio Code.

Last ned VS Code

Plugins til Visual Studio Code

Du vil ha glede av å installere Jon Jayamannes Python plugin til Visual Studio Code

Det gjør du slik:

  1. Start Visual Studio Code
  2. Trykk Ctrl+Shift+P for å åpne VS Codes "kommandopalett"
  3. skriv ext install python og trykk enter
